Anjitha & Shinoj wedding invitation

Open index.html in a browser, or upload this full wedding-invitation folder to any static hosting service.

Editable content:
- config.js controls names, date, time, venue, map link, message, music path, preview image, gallery images, and colors.
- assets/ contains the copied photos and music/video file.

WhatsApp preview:
- The page includes Open Graph preview tags. WhatsApp will show the preview after the folder is hosted at a public HTTPS URL.
- If you change the preview photo, update both previewImage in config.js and the og:image tag in index.html.

Missing asset:
- TMM Auditorium.png was not found at the provided path, so it was not included.
